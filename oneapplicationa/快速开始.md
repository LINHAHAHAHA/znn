# 快速开始指南

## 使用Android Studio打开项目（推荐方式）

由于Gradle Wrapper需要首次下载依赖，建议直接使用Android Studio：

### 步骤：

1. **打开Android Studio**

2. **导入项目**
   - File → Open
   - 选择项目目录：`E:\计算机毕业设计2024\00242基于大数据的公安接处警平台建设与应用研究\oneapplicationa`
   - 点击OK

3. **等待Gradle同步**
   - Android Studio会自动下载Gradle和依赖
   - 首次同步可能需要5-10分钟
   - 确保网络连接正常

4. **连接小米手机**
   - 开启USB调试
   - 连接手机到电脑
   - 在手机上允许USB调试

5. **运行应用**
   - 点击工具栏的绿色运行按钮
   - 选择你的小米手机
   - 应用会自动安装并启动

## 如果没有Android Studio

### 方法1：下载预编译APK（最简单）
如果你只是想测试应用，可以：
1. 让有Android Studio的同学帮你编译APK
2. 直接安装APK到手机

### 方法2：安装Android Studio
1. 下载：https://developer.android.google.cn/studio
2. 安装Android Studio
3. 按照上面的步骤打开项目

## 常见问题

### Q: Gradle同步失败？
A: 
- 检查网络连接
- 配置国内镜像（阿里云）
- 在项目根目录的build.gradle中添加：
```gradle
allprojects {
    repositories {
        maven { url 'https://maven.aliyun.com/repository/google' }
        maven { url 'https://maven.aliyun.com/repository/public' }
        google()
        mavenCentral()
    }
}
```

### Q: 找不到JDK？
A: 
- Android Studio会自动下载JDK
- 或手动下载JDK 17并配置JAVA_HOME

### Q: 手机无法连接？
A: 
- 确保开启USB调试
- 安装手机驱动
- 尝试不同的USB接口
