# 公安接处警平台 - 部署指南

## 🚀 快速生成APK（3种方法）

### ⭐ 方法一：双击运行脚本（最简单）

1. **双击运行**：`生成APK.bat`
2. 等待编译完成（首次需要10-15分钟）
3. 自动打开APK所在文件夹
4. 将 `app-debug.apk` 复制到手机安装

**注意**：首次运行需要下载Gradle和依赖（约500MB），请保持网络连接。

---

## 在小米手机上安装运行

### 方法二：使用Android Studio直接安装（推荐）

1. **准备小米手机**
   - 进入"设置" → "我的设备" → "全部参数"
   - 连续点击"MIUI版本"7次，开启开发者模式
   - 返回"设置" → "更多设置" → "开发者选项"
   - 开启"USB调试"和"USB安装"

2. **连接手机到电脑**
   - 使用USB数据线连接手机和电脑
   - 手机上会弹出"允许USB调试"提示，点击"确定"
   - 在电脑命令行输入：`adb devices`
   - 确认能看到你的设备

3. **在Android Studio中运行**
   - 打开项目：`/e:/计算机毕业设计2024/00242基于大数据的公安接处警平台建设与应用研究/oneapplicationa`
   - 等待Gradle同步完成
   - 点击工具栏的运行按钮（绿色三角形）
   - 在设备列表中选择你的小米手机
   - 应用会自动安装并启动

### 方法三：使用命令行生成APK

1. **打开命令行**
   - 在项目目录按住Shift，右键点击空白处
   - 选择"在此处打开PowerShell窗口"

2. **执行命令**
   ```bash
   .\gradlew.bat assembleDebug
   ```

3. **找到APK**
   ```
   app\build\outputs\apk\debug\app-debug.apk
   ```

### 方法四：使用Android Studio生成签名APK（发布版）

1. **生成签名密钥**
   - Android Studio → Build → Generate Signed Bundle/APK
   - 选择APK → Next
   - 点击"Create new..."创建新密钥
   - 填写密钥信息并保存

2. **生成签名APK**
   - 选择刚创建的密钥
   - 选择"release"构建类型
   - 点击Finish
   - APK生成在：`app/release/app-release.apk`

## 首次运行配置

1. **授予权限**
   - 首次启动会请求"录音权限"
   - 点击"允许"授予权限

2. **网络连接**
   - 确保手机已连接网络（WiFi或移动数据）
   - AI分析功能需要网络连接

3. **测试功能**
   - 点击"开始录音"测试语音识别
   - 输入测试文本，点击"AI智能分析"
   - 保存一条测试记录

## 常见问题

### 1. 无法连接设备
- 检查USB调试是否开启
- 尝试更换USB数据线或USB接口
- 重新安装手机驱动程序
- 执行：`adb kill-server` 然后 `adb start-server`

### 2. Gradle同步失败
- 检查网络连接
- 配置Gradle镜像（阿里云镜像）
- 清理项目：Build → Clean Project

### 3. 语音识别不工作
- 检查录音权限是否授予
- 确保在安静环境下使用
- 检查手机是否支持语音识别

### 4. AI分析失败
- 检查网络连接
- 确认DeepSeek API密钥有效
- 查看日志输出错误信息

### 5. 应用闪退
- 查看Android Studio的Logcat日志
- 检查是否授予所有必要权限
- 尝试卸载重新安装

## 性能优化建议

1. **首次使用**
   - 首次启动可能较慢，需要初始化数据库
   - 首次AI分析可能需要更长时间

2. **网络优化**
   - 建议使用WiFi进行AI分析
   - 移动网络可能较慢

3. **存储空间**
   - 应用大小约20-30MB
   - 数据库会随着记录增加而增长
