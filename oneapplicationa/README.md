# 公安接处警平台 Android 应用

## 项目简介

这是一个基于Android的公安接处警平台应用，集成了AI智能分析功能，可以实现：

1. **语音转文字**：使用Android系统语音识别功能，实时将报警电话语音转为文字
2. **AI智能分析**：集成DeepSeek AI，自动提取时间、地点、人物等关键信息
3. **警情分类**：AI自动判断警情类型（盗窃、纠纷、求助、交通事故等）
4. **紧急程度评估**：智能评估警情紧急程度（1-5级）
5. **处置预案推荐**：根据警情内容推荐处置方案
6. **接警单管理**：保存、查询、更新接警记录

## 技术栈

- **开发语言**：Kotlin
- **架构模式**：MVVM
- **UI框架**：Material Design
- **数据库**：Room
- **网络请求**：Retrofit + OkHttp
- **AI接口**：DeepSeek API
- **语音识别**：Android SpeechRecognizer

## 核心功能

### 1. 语音接警
- 点击"开始录音"按钮进行语音输入
- 系统自动将语音转换为文字
- 支持中文语音识别

### 2. AI智能分析
- 点击"AI智能分析"按钮
- DeepSeek AI自动分析报警内容
- 提取关键信息：
  - 事发地点
  - 事发时间
  - 涉及人员
  - 事件描述
- 判断警情类型和紧急程度
- 推荐处置预案

### 3. 接警单管理
- 自动填写接警单信息
- 支持手动修改和补充
- 保存接警记录到本地数据库
- 查看历史接警记录
- 更新处理状态和结果

### 4. 警情分类
支持以下警情类型：
- 盗窃
- 纠纷
- 求助
- 交通事故
- 火警
- 医疗急救
- 暴力事件
- 诈骗
- 失踪
- 其他

### 5. 紧急程度评估
- 5级（特急）：危及生命、正在发生的暴力犯罪、火灾等
- 4级（紧急）：可能危及安全、财产损失较大
- 3级（中等）：一般治安案件、纠纷
- 2级（较轻）：咨询类、轻微求助
- 1级（一般）：非紧急事务

## 安装和运行

### 环境要求
- Android Studio Arctic Fox 或更高版本
- JDK 17
- Android SDK 34
- 最低支持 Android 7.0 (API 24)

### 安装步骤

1. **克隆或下载项目**
   ```bash
   # 项目已在当前目录
   ```

2. **打开项目**
   - 使用Android Studio打开项目目录
   - 等待Gradle同步完成

3. **配置API密钥**
   - 打开 `DeepSeekApiService.kt`
   - 确认API密钥已配置（已包含在代码中）

4. **连接设备**
   - 连接小米手机到电脑
   - 在手机上开启开发者选项和USB调试
   - 确认设备已连接：`adb devices`

5. **运行应用**
   - 点击Android Studio的运行按钮
   - 选择你的小米手机
   - 等待应用安装和启动

### 生成APK

1. **生成调试版APK**
   ```bash
   ./gradlew assembleDebug
   ```
   APK位置：`app/build/outputs/apk/debug/app-debug.apk`

2. **生成发布版APK**
   ```bash
   ./gradlew assembleRelease
   ```
   APK位置：`app/build/outputs/apk/release/app-release.apk`

3. **安装到手机**
   ```bash
   adb install app/build/outputs/apk/debug/app-debug.apk
   ```

## 使用说明

### 接警流程

1. **启动应用**
   - 打开"公安接处警平台"应用

2. **语音录音**
   - 点击"开始录音"按钮
   - 说出报警内容（例如："我在人民路123号，刚才有人偷了我的手机"）
   - 点击"停止录音"
   - 语音自动转换为文字

3. **AI分析**
   - 点击"AI智能分析"按钮
   - 等待AI分析完成
   - 系统自动填写：
     - 事发地点：人民路123号
     - 警情类型：盗窃
     - 紧急程度：3级
     - 处置预案：建议立即派出民警前往现场...

4. **补充信息**
   - 填写报警人电话和姓名
   - 检查并修改AI提取的信息
   - 添加备注

5. **保存接警单**
   - 点击"保存接警单"按钮
   - 接警记录保存到数据库

6. **查看历史**
   - 点击"查看历史记录"按钮
   - 查看所有接警记录
   - 点击记录查看详情

7. **处理接警**
   - 在详情页面更新处理状态
   - 填写处理人和处理结果
   - 点击"更新"保存

## 测试案例

### 案例1：盗窃案件
**报警内容**：
"警察同志，我是张三，我在建设路88号小区，今天下午3点左右，我的电动车被偷了，是一辆蓝色的雅迪电动车，车牌号是苏A12345。"

**AI分析结果**：
- 地点：建设路88号小区
- 时间：今天下午3点左右
- 人物：张三
- 警情类型：盗窃
- 紧急程度：3级
- 处置预案：立即派出民警前往现场调查，调取监控录像，登记被盗物品信息

### 案例2：纠纷案件
**报警内容**：
"你好，我在解放路商场，有两个人在吵架，好像要打起来了，请你们快点来。"

**AI分析结果**：
- 地点：解放路商场
- 时间：现在
- 警情类型：纠纷
- 紧急程度：4级
- 处置预案：立即派出民警前往现场制止，防止事态升级

### 案例3：求助案件
**报警内容**：
"我是李四，我在人民医院，我妈妈突然晕倒了，需要帮助。"

**AI分析结果**：
- 地点：人民医院
- 人物：李四、李四的妈妈
- 警情类型：医疗急救
- 紧急程度：5级
- 处置预案：立即联系120急救中心，派出民警协助救援
